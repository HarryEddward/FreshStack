- CafeBuy

### Admin Realm
- Los usuarios de ***"CafeBuy Realm"*** se pueden crear a través del administrador de Keycloak.

### CafeBuy Realm

- Los clientes usan OpenID para identificarse y poder cargar los recursos desde los consumidores
